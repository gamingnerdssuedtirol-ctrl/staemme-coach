# Stämme Coach PWA v0.3

## Update
1. ZIP entpacken.
2. Alle Dateien in dein GitHub-Repository `staemme-coach` hochladen und vorhandene Dateien ersetzen.
3. Commit bestätigen.
4. https://gamingnerdssuedtirol-ctrl.github.io/staemme-coach/ neu öffnen und prüfen, ob `v0.3` angezeigt wird.
5. Den neuen Auslese-Befehl kopieren und dein bisheriges Lesezeichen damit ersetzen.

## Normaler Ablauf
1. Im Coach „Spiel öffnen und Daten aktualisieren“ drücken.
2. Im Spiel das Lesezeichen „Coach aktualisieren“ antippen.
3. Übersicht, Hauptgebäude, Kaserne und Stall werden lesend geladen.
4. Der Coach öffnet sich automatisch mit zusammengeführten Daten.
